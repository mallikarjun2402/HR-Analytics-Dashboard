# HR Analytics Dashboard — Employee Attrition & Workforce Intelligence

## Project Overview
An end-to-end HR Analytics Dashboard built using Power BI
to analyze employee attrition patterns and identify
high-risk employees using IBM Watson HR Dataset.

## Business Problem
Organizations lose millions due to employee attrition.
This dashboard helps HR teams identify:
- Which departments have highest attrition
- What factors drive employees to leave
- Which active employees are at high risk
- How compensation impacts retention

## Dashboard Pages
| Page | Description |
|---|---|
| Overview | Executive KPI summary with 8 KPI cards |
| Attrition Analysis | Breakdown by Dept, Age, Gender, Job Role |
| Risk Analysis | Overtime, Satisfaction & Tenure impact |
| Income Analysis | Salary & compensation patterns |

## Key DAX Measures Written
| Measure | Value |
|---|---|
| Total Employees | 1,470 |
| Total Attrition | 237 |
| Attrition Rate % | 16.12% |
| Active Employees | 1,233 |
| Avg Monthly Income | 6,503 |
| Avg Age Attrited | 33.61 |
| Avg Tenure | 7.01 years |
| Overtime Attrition Rate % | 30.53% |
| High Risk Employees | 121 |

## Key Business Insights
- Sales department has highest attrition rate
- Employees working overtime have 3x higher attrition
- Attrition peaks at years 1 and 2 of employment
- Employees with no stock options leave more frequently
- Low job satisfaction + poor work life balance = high risk

## Tools Used
- Power BI Desktop
- DAX (12 Advanced measures)
- IBM Watson HR Analytics Dataset

## Dataset
- Source: IBM Watson Analytics
- Records: 1,470 employees
- Features: 32 columns
